# rain-util-cli

## Introduction

Auxiliary tool

### Instructions for use

#### 1. Install rain-util-cli

```shell
npm install rain-util-cli -g
` ` `

#### 2. Use

<1> Initialize the rbj configuration file

```shell
rain-tool
` ` `

<2> Complete the rbj configuration file

```shell
rain-tool -c
` ` `
