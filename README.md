# rain-util-cli

## 介绍

辅助工具, 现仅支持 [rain-interface-tools](https://github.com/quiet-rain-ky/rain-interface-tools) 辅助使用

### 使用说明

#### 1. 安装 rain-util-cli

```shell
npm install rain-util-cli -g
```

#### 2. rbj 辅助使用命令

<1> 初始化 rbj 配置文件

```shell
rbj-tool -i
```

<2> 补全 rbj 配置文件

```shell
rbj-tool -c
```
